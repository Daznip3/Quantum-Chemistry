# Beer Dataset Analysis & ETL Pipeline

Проект для анализа датасетов о пиве с ETL-пайплайном. Один датасет загружен с Google Drive, второй спарсен с сайта.

## О проекте

Комплексное решение для анализа данных о пиве, включающее:

- **ETL-пайплайн** для обработки двух источников
- **Очистку и объединение** датасетов
- **Анализ характеристик** пива (стиль, ABV, IBU, рейтинги)
- **Визуализацию** распределений и тенденций

## Быстрый старт

### Установка и запуск

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/danzip3/beer.git
cd beer-analysis
```
2. **Установите зависимости:**
```
pip install -r requirements.txt
```

3. **Запустите полный ETL-пайплайн:**
```
python3 src/main.py --file-id 1GJeIPS09D5oTesxDLniuoRadnaLvVx8N
```

4. **Для анализа откройте Jupyter Notebook:**
```
jupyter notebook
```

Запустите файл notebooks/kaggle_beer_datasets.ipynb

## ETL Пайплайн
Этап 1: Extract (Извлечение)
```
Загрузка первого датасета с Google Drive
Парсинг второго датасета с сайта
Сохранение в .csv
```
Этап 2: Transform (Трансформация)
```
Приведение типов данных
Нормализация колонок
Обработка пропусков и дубликатов
Объединение датасетов
```
Этап 3: Load (Загрузка)
```
Сохранение в CSV
Загрузка в БД
```
